# 🩸 AI Blood Infection Detection System

An AI-powered web application that analyzes patient data to detect blood infection risk and provides actionable insights through an interactive dashboard.

---

## 🚀 Overview

This project is designed to assist in early detection of blood infections by analyzing medical parameters using data analysis techniques. It provides a user-friendly interface to visualize data and generate insights for better decision-making.

---

## ✨ Key Features

- 🧠 AI-based infection risk prediction
- 📊 Interactive dashboard with real-time analytics
- 📈 Data visualization using charts and graphs
- 🔄 Data preprocessing and validation
- 📄 Report generation functionality
- 🔐 User authentication system
- 🎨 Clean and responsive UI

---

## 🛠️ Tech Stack

- **Python**
- **Streamlit**
- **Pandas & NumPy**
- **Plotly**
